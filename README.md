# Citrix Workspace Manager

A Homebrew tap for [citrix-workspace-manager](https://github.com/NickJLange/osx-citrix-manager-cli) (`cwm`) â€” safely park and unpark Citrix Workspace on macOS without rebooting.

## Installation

### Step 1: Add the tap

```sh
brew tap 5L-Labs/citrix-cli
```

### Step 2: Install the formula

```sh
brew install citrix-workspace-manager
```

### Or as a one-liner

```sh
brew install 5L-Labs/citrix-cli/citrix-workspace-manager
```

## Usage

```sh
cwm --help
```
